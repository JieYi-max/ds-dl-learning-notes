## 整体知识链

```
全连接网络处理图片 → 参数爆炸（100×100→10⁷参数）
  ↓
观察 1：特征只需看局部（感受野）
观察 2：相同特征出现在不同位置（参数共享）
观察 3：下采样不影响识别（池化）
  ↓
CNN = 卷积层（特征提取）+ 池化层（压缩尺寸）+ 全连接层（分类）
```
