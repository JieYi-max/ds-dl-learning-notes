# BN 为什么有效 & 局限

**优先级：⭐⭐ 重要（理解层面）**
**课件：** `normalization_v4.pdf` 第 10-12 页

---

💡 **一句话：BN 让 error surface 平滑，训练更快、更稳定。但它依赖 batch size，大模型改用 Layer Norm。**

---

## BN 为什么有效

### 传统观点：Internal Covariate Shift

每层参数更新后，上一层输出的分布变了，下一层要不断适应新分布。
BN 让每层输出统计量稳定，下一层面对更稳定的输入。

### 当前观点：平滑 Error Surface（2018 年论文）

BN 的真正作用不是解决 Internal Covariate Shift，而是改变了 error surface 的形状。

- 没有 BN：地形崎岖，梯度方向变化剧烈 → 需要小学习率
- 有 BN：地形平滑，梯度方向稳定 → 可以用大学习率

---

## BN 的三个好处

- **训练更快** — 梯度更稳定，可以用更大的学习率
- **对初始化不敏感** — 初始化不好，BN 也能把输出拉回合理范围
- **轻微正则化** — 每个 batch 的 μ、σ 有随机噪声

---

⚠️ **局限 1：依赖 batch size**

batch ≥ 32：μ、σ 估计准确 → BN 效果好

batch ≤ 8：μ、σ 波动大 → BN 效果差

batch = 1：μ 就是自己 → BN 完全失效

⚠️ **局限 2：训练和推理行为不同**

训练用 batch 统计量，推理用全局统计量，模型在两种模式下运行。

---

## 替代方案：Layer Normalization

GPT / LLaMA / DeepSeek 全部用 Layer Norm，不用 BN：

- **BN**：对 batch 维度做归一化 → 依赖 batch size → 适合 CNN
- **LN**：对特征维度做归一化 → 不依赖 batch size → 适合 Transformer

大模型 batch size 很小（1~16），BN 效果差，所以用 LN。

---

## 其他归一化方法

- BatchNorm — 对 batch 归一化，CNN 大 batch
- LayerNorm — 对特征归一化，Transformer
- InstanceNorm — 对单样本单通道，风格迁移
- GroupNorm — 分组归一化，小 batch 视觉任务
