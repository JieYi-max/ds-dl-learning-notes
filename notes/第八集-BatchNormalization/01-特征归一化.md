# 特征归一化（Feature Normalization）

**优先级：⭐ 重要**

## 一句话

> **把不同尺度的特征拉到同一范围，让模型训练更稳定、更快。**

## 为什么需要

不同特征的数值范围差异会导致问题：

```
特征1：面积（80 ~ 200）     ← 数值大
特征2：房间数（2 ~ 5）      ← 数值小

模型会错误地认为"面积"更重要，因为它的数值大，对 loss 的影响也大
```

## 公式

![特征归一化公式](./特征归一化公式.png)

其中：

| 符号 | 含义 |
|---|---|
| $x_i^r$ | 第 r 个样本的第 i 个特征 |
| $m_i$ | 所有样本在第 i 个特征上的均值 |
| $\sigma_i$ | 所有样本在第 i 个特征上的标准差 |
| $\tilde{x}_i^r$ | 归一化后的值 |

## 效果

```
归一化前：loss 地形崎岖，梯度更新走 Z 字形
归一化后：loss 地形更接近圆形，梯度直指最低点
```
