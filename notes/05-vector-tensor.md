# 向量与张量

## 向量（Vector）是什么

**向量 = 按顺序排列的一堆数字**

```
[3.2, -1.5, 0.8, 4.0]   ← 4维向量
```

## 维度对比

```python
标量（0维）：tensor(5)              shape=[]
向量（1维）：tensor([1,2,3])        shape=[3]
矩阵（2维）：tensor([[1,2],[3,4]])  shape=[2,2]
张量（3+维）：多个矩阵堆叠           shape=[batch, ...]
```

## 向量在深度学习里表示什么

| 数据 | 表示形式 |
|---|---|
| 一张 28×28 灰度图 | 784 维向量 |
| 一个词语"猫" | 词向量 [0.3, -0.1, 0.8, ...] |
| 一个样本的特征 | 10 维特征向量 |
| 一个 batch 的数据 | [32, 784] 32个784维向量 |

## 跟之前学的内容关联

```
Momentum 里的 v 和 g 都是向量
v_new = μ×v_old - lr×g   ← 向量的方向运算

一个 batch: [32, 784]
  → 32个样本，每个样本是一个 784 维向量
```
