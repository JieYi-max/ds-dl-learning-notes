# Adam 优化器

## 核心理解

**Adam = Momentum（管方向）+ RMSProp（管步长）+ 偏差修正**

| 成分 | 作用 | 默认值 |
|---|---|---|
| Momentum 方向 | 用历史方向修正当前方向 | β₁=0.9 |
| 自适应学习率 | 每个参数有自己的lr，频繁更新的参数lr自动变小 | β₂=0.999 |
| 偏差修正 | 解决训练初期估计不准的问题 | — |

## 数学公式

```
m_t = β₁·m_{t-1} + (1-β₁)·g_t          ← 动量（梯度滑动平均）
v_t = β₂·v_{t-1} + (1-β₂)·g_t²         ← RMS（梯度平方滑动平均）

m̂_t = m_t / (1 - β₁^t)                  ← 偏差修正
v̂_t = v_t / (1 - β₂^t)                  ← 偏差修正

θ_t = θ_{t-1} - lr / (√v̂_t + ε) × m̂_t  ← 更新
```

## 代码

```python
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
# 绝大多数项目直接用默认参数，只调 lr
```

## 跟其他优化器对比

| 优化器 | 方向怎么走 | 步长怎么调 |
|---|---|---|
| SGD | 只看当前梯度 | 所有参数共用 lr |
| SGD+Momentum | 带历史方向 | 所有参数共用 lr |
| RMSProp | 只看当前梯度 | 每个参数自适应 lr |
| Adam ✅ | 带动量方向 | 每个参数自适应 lr |
